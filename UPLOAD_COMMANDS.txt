# GitHub Upload Script for GIGACHAD Fitness
# Run these commands after creating your GitHub repository

# Step 1: Add your GitHub repository as remote (replace YOUR_USERNAME)
git remote add origin https://github.com/YOUR_USERNAME/gigachad-fitness.git

# Step 2: Push your code to GitHub
git push -u origin main

# Step 3: Enable GitHub Pages
# Go to your repository on GitHub → Settings → Pages
# Source: Deploy from a branch
# Branch: main
# Folder: / (root)
# Save

# Your website will be live at:
# https://YOUR_USERNAME.github.io/gigachad-fitness
